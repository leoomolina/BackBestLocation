# API do Projeto BestLocation

> A BestLocation é uma startup que visa solucionar problemas relacionado a busca e venda de imóveis.





Focada em encontrar o melhor imóvel para o consumidor, a BestLocation é um projeto com o objetivo de ajudar o consumidor que esteja a procura 
de novos imóveis para fazer negócios, encontrar o que precisa baseado no perfil dele.




Além disso, impulsionar também as vendas de corretores e imobiliárias.
